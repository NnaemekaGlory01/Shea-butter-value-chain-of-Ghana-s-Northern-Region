# Shea Butter Value Chain - Northern Ghana

This repository contains Python Jupyter Notebook(s) analyzing the shea butter value chain in Ghana’s Northern Region. The analysis focuses on data cleaning, exploratory data analysis, and visualization to provide insights into production, processing, and market dynamics.

## Repository Structure

- `Python/` – Jupyter Notebook and Python scripts for analysis
- `Data/` – Sample datasets used for the analysis (optional)

## Tools Used

- Python 3.x
- pandas, numpy, matplotlib, seaborn
- Jupyter Notebook
